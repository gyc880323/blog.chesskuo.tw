📃 Nonsense

反組譯後得到的 main

#15 我們知道他會去檢查一個東西
通過後會將儲存第二個輸入的變數拿去執行

那個 func 其實只是在檢查你的輸入中是否存在一段 wubbalubbadubdub 這個字串
而且在檢查期間不能有 ascii < ' ' 的 byte 出現

那就在檢查字串前塞些 alphanumeric shellcode
再 jmp 到 wubba 後面的任意 shellcode 即可
注意 jump 的 byte 也是不能是小於空白 ascii 的
稍微填充一下應該不難
